# Hi there 👋

I am **Parisa Mohammadi**, an Electrical Engineering student at the University of Tehran.  
My interests are mainly around **signal processing**, **compressed sensing**, **machine learning**, **learning theory**, and **efficient neural-network systems**.

I like projects that connect mathematical ideas with implementation: sparse recovery algorithms, inverse problems, generative models, language-model interpretability, optimization, and custom PyTorch/MATLAB experiments.

## Selected Research and Implementation Projects

| Project | Topic | Short Description |
|---|---|---|
| [Adaptive Sparse Language Model](https://github.com/parisainthestars/adaptive-sparse-language-model) | Compressed sensing + custom PyTorch NN | A compact causal language-model prototype with task-conditioned measurements, adaptive sparse supports, prompt compression, and explicit sparse masks over attention heads and feed-forward blocks. |
| [Training Language Models to Explain Their Own Computations](https://github.com/parisainthestars/Training-Language-Models-to-Explain-Their-Own-Computations) | LLM interpretability | Reproducibility study on feature description, activation patching, and input ablation for explaining internal language-model computations. |
| [Piecewise Sparse Network Control](https://github.com/parisainthestars/piecewise-sparse-network-control) | Sparse control + compressed sensing | Sparse control of a network dynamical system using OMP, Piecewise OMP, and Piecewise Sparse Bayesian Learning. |
| [DDPM Compressed Sensing on MNIST](https://github.com/parisainthestars/ddpm-compressed-sensing-mnist) | Generative priors for inverse problems | A PyTorch implementation of compressed sensing with a UNet-based DDPM prior, random Gaussian measurements, and ISTA/L1 baseline. |
| [Boolean Compressed Sensing in MATLAB](https://github.com/parisainthestars/boolean-compressed-sensing-matlab) | Group testing | MATLAB implementation of COMP, Definite Defectives, Boolean Matching Pursuit, and non-negative LASSO for sparse binary recovery. |
| [Sparse Recovery in MATLAB](https://github.com/parisainthestars/sparse-recovery-matlab) | Classical compressed sensing | MATLAB demos for L1 recovery, OMP, DCT/FFT sparsity, speech/image reconstruction, and time-frequency diagnostics. |
| [Compressed Sensing Assignments](https://github.com/parisainthestars/compressed-sensing-assignments) | Course assignments | Jupyter implementations of sparse recovery, greedy pursuit algorithms, sensing matrices, sparse image recovery, and sparse representation classification. |
| [Targeted Forgetting Generative Models](https://github.com/parisainthestars/Targeted-Forgetting-Generative-Models) | Machine unlearning | CVAE-based targeted forgetting, generative replay, latent-space visualization, and adversarial recovery experiments. |
